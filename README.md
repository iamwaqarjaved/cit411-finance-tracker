# CIT 411 Finance Tracker

## Overview

This project is a command-line finance tracker built with Python. The program allows users to enter income and expense transactions, validates the input, calculates a running balance, and prints a formatted ledger summary.

## Features

- Records income transactions
- Records expense transactions
- Calculates a running balance
- Rejects negative or zero amounts
- Rejects empty descriptions
- Handles non-numeric amount input using try/except
- Prints a formatted transaction ledger using f-strings

## Requirements

- Python 3.12 or higher
- VS Code
- Python extension for VS Code
- Pylance extension for VS Code
- Git
